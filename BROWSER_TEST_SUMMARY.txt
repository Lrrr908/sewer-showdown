COMMON ISSUES AND SOLUTIONS
--------------------------------------------------------------------------------

Issue: Blank black screen, no map visible
Solution: Check console for sprite loading errors

Issue: Console shows "Failed to load URL" errors
Solution: Verify all sprite files exist in sprites/ directory

Issue: "CORS policy" errors
Solution: Make sure you're accessing via http://localhost:8080, not file://

Issue: Controls don't work
Solution: Click on the canvas first to give it focus

Issue: Game is very slow
Solution: Try a different browser or check CPU usage
